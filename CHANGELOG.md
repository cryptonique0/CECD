# Changelog

All notable changes to the CECD project will be documented in this file.

## [2.0.0] - 2025-12-02

### Added
- **Stable Coin Donation System**: Support for CELO, cUSD, and cEUR donations
- **Volunteer Reputation System**: On-chain reputation tracking with badges
- **Offline Support**: PWA capabilities with service worker
- **Multi-Signature Wallet**: Emergency fund management with proposals
- **AI Incident Prediction**: ML-powered risk analysis
- **Real-Time Notifications**: Live update system with WebSocket support
- **Multi-Language Support**: English, Spanish, and French translations
- **Language Switcher**: Easy language selection in header
- **PWA Manifest**: Mobile app installation support
- **Service Worker**: Offline caching strategy

### Changed
- Enhanced Dashboard with new components
- Updated Header with language switcher
- Improved notification system with animations

### Technical
- Added i18next for internationalization
- Integrated ERC20 token transfers
- Implemented localStorage sync for offline mode
- Added slide-in animations for notifications
